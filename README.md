# YouTube One-Command Toolkit

A simple Bash script to download audio or video from YouTube using one command.

## Features
- Download audio (mp3 or m4a)
- Download full video (mp4)
- Uses yt-dlp + ffmpeg
- Handles cookies automatically

## Requirements
- yt-dlp
- ffmpeg
- Firefox

## Usage
```bash
./yt-tool.sh audio <url>
./yt-tool.sh m4a   <url>
./yt-tool.sh video <url>
